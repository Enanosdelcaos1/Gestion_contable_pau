# Gestión Contable Familiar

Este proyecto es una aplicación web sencilla para gestionar los ingresos y gastos familiares, mostrando el balance neto en tiempo real.

## Características
- Registro de movimientos (ingresos y gastos) mediante formulario.
- Visualización de los totales de ingresos, gastos y balance neto.
- Tabla dinámica con los movimientos realizados.
- Interfaz clara y moderna.

## Estructura del proyecto
```
index.html
css/
  style.css
  reset.css
js/
  gestion.js
```

## Cómo usar
1. Abre `index.html` en tu navegador.
2. Completa el formulario con el concepto, importe y tipo de movimiento.
3. Haz clic en "Registro movimiento" para añadir el movimiento.
4. Los totales y el balance se actualizan automáticamente.

## Requisitos
- Navegador web moderno (Chrome, Firefox, Edge, etc.)
